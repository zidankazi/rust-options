Design Principles

Speed is a feature

Sub-microsecond pricing, zero allocations in hot paths. The math is never the bottleneck.

No unnecessary dependencies

Normal CDF, PRNG, and RNG implemented from scratch for WASM portability.

Pure math in the pricer

No IO, no async, no side effects. Inputs in, prices out.

Correct first, then fast

Validated against known analytical results before optimizing.
